# Android_App_Development_Using_Gen_Ai-KumbaraKala

### AI Powered Artisan Promotion App

Kumbara-Kala is an Android application developed using **Kotlin** and **Jetpack Compose** that helps traditional clay artisans digitally promote and showcase their handmade products. The application uses **Generative AI** to create attractive product descriptions and storytelling content that artisans can easily share on social media platforms.

---

# 📌 Problem Statement

Traditional potters and clay artisans are gradually losing business due to competition from plastic products and lack of digital marketing knowledge. Many artisans are unable to communicate the health, eco-friendly, and cultural value of clay products to modern customers.

Kumbara-Kala bridges this gap by providing a modern Android platform that combines traditional craftsmanship with AI-powered digital promotion.

# ⚙️ Installation

## Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/kumbara-kala.git
```

---

## Step 2: Open in Android Studio

* Open Android Studio
* Click "Open Project"
* Select the cloned project folder

---

## Step 3: Sync Gradle

Allow Gradle to download dependencies.

---

## Step 4: Run Application

* Connect Android device or start emulator
* Click ▶ Run
